# Actividad N°5: POO en Python - Cliente de Correo

**Institución:** UNSAdA (Universidad Nacional San Antonio de Areco)
**Asignatura:** Laboratorio de Lenguajes
**Actividad:** Actividad N°5 - POO en Python

## Descripción del Proyecto

Este proyecto es una aplicación desarrollada en Python utilizando el paradigma de Programación Orientada a Objetos (POO). El objetivo principal es modelar un cliente de correo electrónico básico, inspirado en sistemas como Outlook Express o Thunderbird.

El sistema administra una única cuenta de correo electrónico y una agenda de contactos, permitiendo organizar los mensajes en carpetas de "recibidos" y "enviados".
